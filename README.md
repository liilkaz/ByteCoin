# ByteCoin

* Проект на Swift 5, минимальная поддерживаемая iOS – 15,
* Верстка на UIKit кодом.
* API - https://www.coinapi.io/

---

Приложение предоставит вам последние цены на биткойны во всех популярных, торгуемых валютах.
